二叉排序树 基础 ​

二叉排序树又称二叉查找树,是一种对排序和查找都很有用的特殊二叉树;在二叉排序树中,当前根节点的左子树的结点的值比当前根节点小,当前根节点的右子树的结点的值比当前根节点大;

数据元素类型 ​

二叉排序树的二叉链表存储表示;

c
typedef struct BSTNode{
    int key;
    struct BSTNode* left;
    struct BSTNode* right;
}BSTNode,*BSTree;

二叉排序树查找 ​

二叉排序树可以看成一个有序表,所以在二叉排序树上进行查找和折半查找类似,也是一个逐步缩小查找范围的过程;

c
BSTNode* SearchBST(BSTNode* root,int key){
    if((root==NULL)||root->key==key)return root;
    else if(root->key<key)return SearchBST(root->right,key);
    else if(root->key>key)return SearchBST(root->left,key);
};

二叉排序树的插入 ​

二叉排序树的插入操作是基于查找,要将一个关键字为key的结点插入二叉排序树中,则需要从根结点向下查找,当树中不存在的时候添加一个叶子结点;

c
void InsertBST(BSTree* BT,int key){
    if(*BT==NULL){
        BSTNode* S=(BSTNode*)malloc(sizeof(BSTNode));
        S->key=key;
        S->left=S->right=NULL;
        *BT=S;
    }else if(key<(*BT)->key){
        InsertBST((&(*BT)->left),key);
    }else if(key>(*BT)->key){
        InsertBST(&((*BT)->right),key);
    }
};

二叉排序树删除 ​

被删除的结点可能是二叉排序树中的任何结点,删除结点后,要根据其位置不同修改其双亲结点及相关结点的指针,以保持二叉排序树的特性;

c
void DeleteBST(BSTree *T,int key){
    BSTNode* p=*T;
    BSTNode* f=NULL;
    while(p!=NULL){
        if(p->key==key)break;
        f=p;
        if(p->key<key)p=p->right;
        else if(p->key>key)p=p->left;
    }
    if(p==NULL)return;
    BSTNode* q=p;
    if((p->left!=NULL)&&(p->right!=NULL)){
        BSTNode* s=p->left;
        while(s->right!=NULL){
            q=s;
            s=s->right;
        };
        p->key=s->key;
        if(q!=p)q->right=s->left;
        else if(q==p)q->left=s->left;
        free(s);
        return;
    }else if(p->right==NULL){
        p=p->left;
    }else if(p->left==NULL){
        p=p->right;
    }
    if(!f)*T=p;
    else if(q==f->left)f->left=p;
    else if(q==f->right)f->right=p;
    free(q);
};
